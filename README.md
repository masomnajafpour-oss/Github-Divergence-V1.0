# MetaTrader 5 Divergence Indicator (V1.0)

یک اندیکاتور پیشرفته برای MetaTrader 5 که سیگنال‌های خرید و فروش بر اساس چهار نوع واگرایی را شناسایی می‌کند.

## ویژگی‌ها

### انواع سیگنال‌ها (4 نوع)
1. **دو واگرایی معمولی مثبت پشت‌سرهم** → سیگنال خرید (فلش سبز)
2. **دو واگرایی معمولی منفی پشت‌سرهم** → سیگنال فروش (فلش قرمز)
3. **واگرایی معمولی مثبت + واگرایی مخفی منفی** → سیگنال فروش (فلش قرمز)
4. **واگرایی معمولی منفی + واگرایی مخفی مثبت** → سیگنال خرید (فلش سبز)

### الگوهای کندلی
- **Pin Bar**: سایه طولانی با بدن کوچک
- **Engulfing**: کندل بزرگ‌تر که کندل قبلی را احاطه کند

### ویژگی‌های رسم‌شده
- **دو خط RSI**: یکی قرمز (براساس Low) و یکی آبی (براساس High)
- **خطوط پیوت**: خطوط چین برای اتصال سه پیوت
- **فلش سیگنال**: سبز برای خرید، قرمز برای فروش

## تنظیمات

### فعال/غیرفعال کردن انواع واگرایی
- `Enable_DoubleBullish`: دو واگرایی معمولی مثبت
- `Enable_DoubleBearish`: دو واگرایی معمولی منفی
- `Enable_BullishThenHiddenBearish`: معمولی مثبت + مخفی منفی
- `Enable_BearishThenHiddenBullish`: معمولی منفی + مخفی مثبت

### الگوهای کندلی
- `Enable_PinBar`: فعال/غیرفعال الگوی Pin Bar
- `Enable_Engulfing`: فعال/غیرفعال الگوی Engulfing

### تنظیمات RSI
- `RSI_Period`: دوره RSI (پیش‌فرض: 14)
- `Overbought`: سطح اشباع خرید (پیش‌فرض: 70)
- `Oversold`: سطح اشباع فروش (پیش‌فرض: 30)

### تنظیمات واگرایی
- `PivotStrength`: تعداد کندل‌های آخر برای بررسی آخرین پیوت (پیش‌فرض: 3)
- `CycleCandles`: حداکثر فاصله بین پیوت 1 تا 3 (پیش‌فرض: 111)
- `SignalLookForward`: تعداد کندل‌های بعد از الگوی کندلی برای رسم سیگنال (پیش‌فرض: 5)
- `PinBarRatio`: نسبت سایه به بدن در الگوی Pin Bar (پیش‌فرض: 2.5)

### فاصله سیگنال‌ها
- `Signal_Distance_E0_E1`: فاصله بین اولین و دومین سیگنال (points)
- `Signal_Distance_Ei_Ei1`: فاصله بین سیگنال‌های متوالی برای i≥1 (points)

## نحوه استفاده

1. فایل `DivergenceIndicator.mq5` را در فولدر `MQL5/Indicators` MetaTrader 5 قرار دهید
2. MetaTrader 5 را راه‌اندازی کنید
3. Indicator را در نمودار بارگذاری کنید
4. تنظیمات را براساس نیاز خود تنظیم کنید
